## 📊 Task 1: Titanic Survival Analysis
**Objective:** Identify the patterns that dictated survival on the Titanic.
- **Key Insight:** Survival was heavily biased toward female passengers and those in 1st class.
- **Link:** [View Task 1 Notebook](./Task1_EDA.ipynb)

## 🧹 Task 2: Data Cleaning & Preprocessing
**Objective:** Transform raw data into a clean, model-ready format.
- **Key Techniques:** Handling missing values, encoding categorical variables, and outlier detection using the IQR method.
- **Link:** [View Task 2 Notebook](./Task2_Data_cleaning_and_preprocessing.ipynb)

## 📈 Task 3: Comprehensive EDA & Distribution
**Objective:** A deep-dive into the dataset's structure, data types, and numerical distributions.
- **Key Features:**
